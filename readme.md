# Torre test app
app created with ionic/angular

## Setup 
Install ionic cli:
- `npm install -g @ionic/cli`

## clone the repository
https://github.com/Hefziben/amasohn-store.git

## install the dependencies
- run `npm install` 

## Running the app 
-run `ionic serve` or `ng serve` 

## Testing
-run `npm test`
